= PaginationFu

Pagination that is fast, simple and fun!

== Installation

There are two ways to start using pagination_fu.
The Gem:

	gem install grzegorzkazulak-pagination_fu --source http://gems.github.com/
	
or the plugin

	script/plugin install git://github.com/grzegorzkazulak/pagination_fu.git
	
That's it. You are ready to roll.
